# 투어룩 (Tour look)

## 📖 프로젝트 개요
 - 각 지역의 여행지를 자유롭게 공유하는 커뮤니티 입니다.

- 배포 : https://chiz.pythonanywhere.com
---

## stack
- HTML
- CSS
- Javascript
- Python 3.8
- Flask
- SQLite

## 🌟 주요 기능

- 회원가입
- 로그인 / 로그아웃
- 게시글 등록
- 게시글 리스트 & 상세보기
- 댓글 등록
- 지역별로 여행지 게시글 리스트 검색
